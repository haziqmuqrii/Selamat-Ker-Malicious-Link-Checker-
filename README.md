# Selamat-Ker-Malicious-Link-Checker-
FYP Project

📌 Project Overview

SelamatKer is a mobile application designed to detect malicious URLs and phishing links to protect users from online scams, fraud, and cyber threats.

The app analyzes URLs using a detection model and classifies them as:

Safe ✅
Suspicious ⚠️
Malicious ❌

Goal: Help users avoid phishing attacks and browse safely.

🎯 Objectives
   
Detect phishing and malicious URLs accurately
Provide instant feedback to users
Increase cybersecurity awareness
Reduce risk of scams and data theft

🧠 System Workflow
User inputs URL
App sends data to backend / model
Model analyzes URL features
Prediction generated
Result displayed to user with confidence score
